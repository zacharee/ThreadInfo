## ThreadInfo

Very simple app to list and export the operational datasets for the preferred and available local Thread networks from your device.

### Usage
1. Launch the app.
2. Accept the prompt to allow access to your Thread credentials.
3. Accept the second prompt to allow access to your Thread credentials.
4. Each available network will be listed. Tap an item to export the dataset as a hexadecimal string. You can then import this in Home Assistant or other platforms.

It may take a few seconds for each prompt to show up. Right now, if Thread isn't available on your device, the app will just crash. Later versions may add proper error messages.
